# Sai Divya weds Sai Ramakrishna

A cinematic, scroll-driven wedding invitation.
**Friday, 3rd July 2026 · 8:12 a.m. · M.N.R. Gardens, Mancherial.**

## Deploy
`index.html` is fully self-contained (all imagery embedded). Host it anywhere,
or enable **GitHub Pages** (Settings → Pages → deploy from `main` /root) and the
invite goes live at the Pages URL — share that link on WhatsApp.

## Source
- `index.html` — the built, standalone invite (this is what you deploy)
- `assets/` — optimized WebP layers (sky, temples, garden, foreground, peacock, cows, couple)
- `build.py` — regenerates `index.html` from the layers (keys backgrounds, embeds, builds markup)

Built with GSAP + ScrollTrigger + Lenis.
